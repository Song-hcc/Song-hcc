# 学生管理系统

## 介绍
语言：C语言
界面：控制台
实现方法：单链表、文件操作

## 功能
1.录入学生信息                 
2.打印学生信息                  
3.保存学生信息                  
4.读取学生信息                  
5.统计学生信息                  
6.查找学生信息                  
7.修改学生信息                  
8.删除学生信息                  
9.学生成绩排名                  
0.  退出系统
